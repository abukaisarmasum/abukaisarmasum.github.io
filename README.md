# GitHub Pages Portfolio (Template)

## What to edit
- Replace `YOUR_USERNAME` in `index.html` with your GitHub username.
- Replace `Your Name`, your email, and your links (Scholar/LinkedIn/CV).
- Update Projects + Publications.

## Optional
- Add `assets/profile.jpg` to show your photo.
- Add `cv.pdf` in the repo root and set the CV link to `cv.pdf`.

## Deploy
GitHub repo Settings → Pages → Deploy from branch → `main` → `/(root)`
